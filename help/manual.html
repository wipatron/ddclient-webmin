Manual IP Update Help

There are five main tabs which serve to group the Manual IP Update options by function. Each option has an explanation to the right. For more information, click on "Search Docs" to the top right of the screen.

The basic operation of the Manual IP Update function is as follows:

  • Configuration Tabs
    Configure the parameters in each of the four tabs as desired. An explanation of the tabs is shown below.
  • Build Button
    Click on the "Build" button to transfer the values you have chosen from the configuration tabs into the command line edit box.
  • Manual Edits
    You can, at any time, manually edit the command line parameters.
  • Test Button
    You can now click on the "Test" button to see if the parameters you have so far entered into the command line, either through manual edits, or using the configuration tabs and clicking on "Build", will work. ddclient is invoked, but with the "-noexec", "-verbose", and "-debug" modes enabled. No action will take place, and you will have a chance to diagnose any problems.
  • Run Button
    When you click on the "Run" button, any parameters currently in the command line edit box are run against the ddclient command. Output is shown immediately below the screen.
  • Save Button
    Once you have a good working set of command line parameters, you can click "save" to save them to a seperate configuration file for future use.
  • Previous Button
    Use this button if you want to retrieve the previous set of parameters.

Here is an explanation of the operation of the configuration tab screens:

  • Usage
    Common options
  • Advanced Usage
    System related functions pertaining to the operation of ddclient.
  • Servers/Protocols
    Select the desired protocol from the dropdown menu. You must then click the "Protocol" button for the options pertinent to that protocol to appear. Note that each protocol has its own specific set of options. Every selection must, however, always include the "server", "login", and "password" options to be filled in.
  • Routers/IP Acquisition
    This tab contains options specific to the method by which your local ddclient will obtain its IP address information. Use the dropdown menu next to "use" to choose this option. Next click on the "Use" button to process your choice. The module will then present you with sub-options specific to the method chosen.
